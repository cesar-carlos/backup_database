# Quick Start

Atalho para o fluxo oficial do instalador:

```powershell
python installer\build_installer.py
```

O comando sincroniza versao, valida ou recompila o build Windows, resolve
dependencias locais do instalador, compila o `setup.iss` e gera o `.sha256`.

Se voce precisa do racional, fallback manual ou detalhes operacionais, use:

- `installer\README.md`
- `docs\install\release_guide.md`
- `installer\TROUBLESHOOTING_SERVICE.md`
